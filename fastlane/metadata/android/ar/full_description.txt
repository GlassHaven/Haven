Haven هو عميل مفتوح المصدر لـ SSH وVNC وRDP وSFTP لأندرويد مع سطح مكتب Wayland أصلي وتخزين سحابي.

الطرفية:
- محاكي VT100/xterm مع Unicode وتبويبات متعددة و6 أنظمة ألوان
- استمرار الجلسة عبر tmux وzellij وscreen وbyobu
- وضع الماوس لتطبيقات TUI (vim وhtop وmc)
- شريط أدوات لوحة مفاتيح قابل للتخصيص مع Esc وCtrl وAlt والأسهم ومفاتيح الوظائف وماكرو مخصص
- تحديد النص مع توسيع الكلمات والنسخ واكتشاف الروابط
- وضع الإدخال الصوتي والإكمال التلقائي
- Mosh وEternal Terminal للاتصالات المتنقلة والمستمرة

سطح مكتب Wayland الأصلي:
- مُركّب labwc مُسرّع بوحدة معالجة الرسومات يعمل مباشرة على جهازك
- لوحة waybar ومُشغّل تطبيقات fuzzel ومدير ملفات thunar ومحرر mousepad
- حزمة البداية مع Firefox وآلة حاسبة وعارض صور
- XWayland لتوافق تطبيقات X11
- إدارة أسطح مكتب متعددة — تشغيل عدة أسطح مكتب في وقت واحد
- لا يحتاج صلاحيات الجذر (PRoot مع Alpine Linux)

سطح مكتب VNC البعيد:
- عارض VNC مع تكبير بالقرص والتمرير ووضع ملء الشاشة
- دعم نفق SSH للاتصالات المشفرة

سطح مكتب RDP البعيد:
- الاتصال بسطح مكتب Windows البعيد وxrdp وGNOME Remote Desktop
- تكبير بالقرص ولوحة مفاتيح مع تعيين مفاتيح المسح

التخزين السحابي:
- أكثر من 60 مزود عبر rclone — Google Drive وDropbox وOneDrive وS3 والمزيد
- مصادقة OAuth عبر المتصفح
- مزامنة المجلدات مع فلاتر وحدود نطاق ترددي
- بث الوسائط إلى VLC عبر خادم HTTP محلي
- خادم DLNA للتلفزيونات الذكية وChromecast

نقل الملفات:
- متصفح SFTP مع رفع وتنزيل وإعادة تسمية وحذف ونقل مجلدات
- تصفح مشاركات ملفات SMB/Windows مع نفق SSH اختياري

الشبكات:
- SSH مع إعادة اتصال تلقائية وتوجيه المنافذ وProxyJump متعدد القفزات
- اكتشاف أقران Tailscale تلقائياً
- دعم شبكة Reticulum عبر rnsh

الأمان:
- توليد مفاتيح SSH (Ed25519 وRSA وECDSA) مع تخزين مشفر
- دعم مفاتيح FIDO2 (ed25519-sk وecdsa-sk) عبر NFC أو USB
- قفل التطبيق بالبصمة مع مهلة قابلة للتخصيص
- بدون إعلانات، بدون تتبع، بدون جمع بيانات

Haven مجاني ومفتوح المصدر (GPLv3).
