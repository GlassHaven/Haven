Haven — клиент с открытым исходным кодом для SSH, VNC, RDP и SFTP для Android с нативным рабочим столом Wayland и облачным хранилищем.

Терминал:
- Эмулятор VT100/xterm с Unicode, вкладками и 6 цветовыми схемами
- Сохранение сессий через tmux, zellij, screen или byobu
- Режим мыши для TUI-приложений (vim, htop, mc)
- Настраиваемая панель клавиатуры с Esc, Ctrl, Alt, стрелками, функциональными клавишами и макросами
- Выделение текста с расширением слов, копированием и обнаружением URL
- Голосовой ввод и автодополнение
- Mosh и Eternal Terminal для мобильных и постоянных подключений

Нативный рабочий стол Wayland:
- Композитор labwc с GPU-ускорением прямо на устройстве
- Панель waybar, лаунчер fuzzel, файловый менеджер thunar, редактор mousepad
- Стартовый пакет с Firefox, калькулятором, просмотрщиком изображений
- XWayland для совместимости с X11-приложениями
- Управление несколькими рабочими столами одновременно
- Не требует root (PRoot с Alpine Linux)

Удалённый рабочий стол VNC:
- Просмотрщик VNC с масштабированием щипком, прокруткой и полноэкранным режимом
- SSH-туннель для зашифрованных подключений

Удалённый рабочий стол RDP:
- Подключение к Windows Remote Desktop, xrdp и GNOME Remote Desktop
- Масштабирование щипком, клавиатура с маппингом сканкодов

Облачное хранилище:
- Более 60 провайдеров через rclone — Google Drive, Dropbox, OneDrive, S3 и другие
- OAuth-аутентификация через браузер
- Синхронизация папок с фильтрами и ограничением скорости
- Стриминг медиа в VLC через локальный HTTP-сервер
- DLNA-сервер для Smart TV и Chromecast

Передача файлов:
- SFTP-браузер с загрузкой, скачиванием, переименованием, удалением и передачей папок
- SMB/Windows-шары с опциональным SSH-туннелем

Сеть:
- SSH с автоподключением, перенаправлением портов и ProxyJump
- Автообнаружение узлов Tailscale
- Mesh-сеть Reticulum через rnsh

Безопасность:
- Генерация SSH-ключей (Ed25519, RSA, ECDSA) с шифрованным хранением
- Аппаратные ключи FIDO2 (ed25519-sk, ecdsa-sk) через NFC или USB
- Биометрическая блокировка с настраиваемым таймаутом
- Без рекламы, без отслеживания, без сбора данных

Haven бесплатен и имеет открытый исходный код (GPLv3).
